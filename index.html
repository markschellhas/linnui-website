Declarative UI framework for Go

LinnUI is a simple library and UI framework for good-looking Go applications, created and maintained by Mark Schellhas in 2025.

If you're looking for a simple, powerful, less verbose GUI for Go - this is it.

Installation:
go get github.com/markschellhas/linnui/ui
